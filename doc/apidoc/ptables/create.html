POST /api/ptables
Create a ptable.

Examples

POST /api/ptables
{
  "ptable": {
    "name": "ptable_test",
    "layout": "d-i partman-auto/disk"
  }
}
201
{
  "ptable": {
    "name": "ptable_test",
    "id": 1011586619
  }
}
