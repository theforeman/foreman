Params

Param name Description
id
required

POST /api/architectures
Create an architecture.

Examples

POST /api/architectures
{
  "architecture": {
    "name": "i386"
  }
}
200
{
  "architecture": {
    "operatingsystem_ids": [],
    "id": 501905020,
    "name": "i386"
  }
}

Params

Param name Description

required

required
Value: Must be String

optional

Value: Must be Array
