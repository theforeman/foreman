POST /api/architectures
Create an architecture.

Examples

POST /api/architectures
{
  "architecture": {
    "name": "i386"
  }
}
200
{
  "architecture": {
    "name": "i386",
    "id": 501905020,
    "operatingsystem_ids": []
  }
}

Params

Param name Description
architecture
required
Value: Must be a Hash
architecture[name]
required
Value: Must be String
architecture[operatingsystem_ids]
optional

Operatingsystem ID's


Value: Must be Array
