GET /api/puppetclasses/:id
Show a puppetclass.

Examples

GET /api/puppetclasses/base
200
{
  "puppetclass": {
    "lookup_keys": [
      {
        "lookup_key": {
          "description": null,
          "default_value": "443",
          "required": false,
          "override": false,
          "id": 298486374,
          "key": "ssl_port",
          "is_param": false
        }
      }
    ],
    "name": "base",
    "id": 980190962
  }
}

Params

Param name Description
id
required
Value: Must be an identifier, string from 2 to 128 characters containting only alphanumeric characters, space, '_', '-' with no leading or trailing space..
>>>

POST /api/puppetclasses
Create a puppetclass.

Examples

POST /api/puppetclasses
{
  "puppetclass": {
    "name": "test_puppetclass"
  }
}
201
{
  "puppetclass": {
    "lookup_keys": [],
    "name": "test_puppetclass",
    "id": 980190963
  }
}

Params

Param name Description
puppetclass
required
Value: Must be a Hash
puppetclass[name]
required
Value: Must be String
