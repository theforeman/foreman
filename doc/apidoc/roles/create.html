POST /api/roles
Create an role.

Examples

POST /api/roles
{
  "role": {
    "name": "staff"
  }
}
200
{
  "role": {
    "permissions": [],
    "builtin": 0,
    "name": "staff",
    "id": 11
  }
}

Params

Param name Description
role
required
Value: Must be a Hash
role[name]
required
Value: Must be String
