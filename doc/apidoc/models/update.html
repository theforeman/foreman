PUT /api/models/:id
Update a model.

Examples

PUT /api/models/980190962
{
  "name": "KVM"
}
200
{
  "model": {
    "id": 980190962,
    "vendor_class": null,
    "hardware_model": null,
    "created_at": "2012-11-11T12:16:14Z",
    "info": "Virtual Machine",
    "updated_at": "2012-11-11T12:16:14Z",
    "name": "KVM"
  }
}

Params

id
required

Value: Must be a Hash

required
Value: Must be String

Value: Must be String

Value: Must be String
